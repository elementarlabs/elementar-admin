Marquee

The @emr-marquee is an Angular marquee library that delivers a versatile marquee component with various configuration options, including animation speed, reverse scrolling, gap size, and the ability to pause on hover.

Key features:
  • ✅ Includes options to control animation speed and reverse scrolling.
  • ✅ Allows the marquee to pause when hovered.
  • ✅ Lightweight and simple to integrate as a standalone component.
