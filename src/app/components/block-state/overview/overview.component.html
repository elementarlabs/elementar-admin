Block state

Got an empty page with no content to show? Block state has you covered! This handy Angular Material component lets you add custom messages, visuals, or placeholders to make those empty states look polished and purposeful.

No more boring "nothing here" vibes—give your users a better experience with just a few lines of code.

Key features:
  • ✅ Super easy to set up and customize.
  • ✅ Works great with Angular Material and Tailwind.
  • ✅ Responsive and user-friendly right out of the box.
  • ✅ Perfect for dashboards, lists and empty screens.
